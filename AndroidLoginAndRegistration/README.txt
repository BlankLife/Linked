****These are the steps for getting your local server to work with the login and registration feature for Linked.
1. Install xampp-portable-win32-5.6.24-1 on your drive.
2. Put the android_linked_login_api folder into the /xampp/hdocs directory.
3. Put the android_linked_api folder into the /xampp/mysql/data directory. 
4. Run the xampp-control exe and on the Xampp control panel click 'Start' under the Actions for both Apache and MySQL.
5. Once both Apache and MySQL are runnning, your local server is running.
6. Open AndroidLoginAndRegistration with android studio, and build and run it!
